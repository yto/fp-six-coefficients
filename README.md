# FPの6つの係数

FP（ファイナンシャルプランニング）で使われる6つの係数を、グラフと表でインタラクティブに学べるツールです。

**公開URL:** https://yto.github.io/fp-six-coefficients/

---

## 6つの係数

| 係数名 | 計算式 | 概要 | タグ |
|---|---|---|---|
| 終価係数 | `(1+r)ⁿ` | 現在の1円が将来いくらになるか | 一括・将来 |
| 現価係数 | `1/(1+r)ⁿ` | 将来の1円は現在いくらか | 一括・現在 |
| 年金終価係数 | `[(1+r)ⁿ-1]/r` | 毎年1円積み立てると将来いくらになるか | 積立・将来 |
| 減債基金係数 | `r/[(1+r)ⁿ-1]` | 将来1円を貯めるために毎年いくら積み立てるか | 積立・現在 |
| 資本回収係数 | `r/[1-(1+r)⁻ⁿ]` | 今1円あると毎年いくら受け取れるか | 取崩・将来 |
| 年金現価係数 | `[1-(1+r)⁻ⁿ]/r` | 毎年1円受け取るために今いくら必要か | 取崩・現在 |

### 係数の分類

```
          [将来] 現在→将来    [現在] 将来→現在
[一括]    終価係数            現価係数
[積立]    年金終価係数        減債基金係数
[取崩]    資本回収係数        年金現価係数
```

## 使い方

### 1. 係数タブを選ぶ

画面上部の6つのタブから確認したい係数を選択します。

### 2. スライダーを操作する

- **年利率**（0.1〜20%）
- **運用期間**（1〜50年）

スライダーを動かすと、グラフ・係数値・テーブルがリアルタイムで更新されます。

### 3. グラフを読む

各係数に合わせた積み上げ棒グラフが表示されます。

- **青**: 元本・積立累計・元金残
- **緑**: 運用益

バーにマウスオーバーすると、各値と合計がツールチップで確認できます。

### 4. テーブルを読む

画面下部のテーブルには、1年目から運用期間の最終年まで、6つの係数の値が一覧で表示されます。現在選択中の係数列がハイライトされます。

## ファイル構成

```

├── index.html       # メインページ（HTML/CSS/JS 単一ファイル）
├── manifest.json    # PWA設定
├── sw.js            # Service Worker（オフライン対応）

```

```bash
cd fp-six-coefficients
python3 -m http.server 3000
# → http://localhost:3000 を開く
```
